WASM Grid Game
4-way splitscreen AI simulation in C++/WebGL/WebAssembly.

Build: Install Emscripten SDK, run build.bat
Run: python -m http.server 8000, open http://localhost:8000

src/cpp/ - C++ source
src/js/ - WASM loader  
build/ - Compiled output
index.html - Entry point

Requires: Emscripten SDK, WebGL 2.0 browser
